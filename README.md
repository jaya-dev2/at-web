# Project Setup  

To run the server locally:  

1. Navigate to the server folder:  
   ```bash
   cd server
2. Install dependencies
   ```bash
   npm install
3. Rename .env.test file to .env
   ```bash
   mv .env.test .env  
4. Run service in localhost
   ```bash
   npm run test
5. Navigate to http://localhost/ in your web browser
